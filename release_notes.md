# 🌟 NeoQN - The fully Overhauled UI and Bug Fixes Update! 🪄 (v2.0.9)

🚨 **Important Notes Before Updating:**

*   **For QuickNovel Users:** Please create a backup of your data in QuickNovel before switching. You can effortlessly restore this backup in NeoQN!
*   **For Existing NeoQN Users:** Due to the updated application signature and package name changes, **this version will install as a separate app** alongside your older release. Please create a backup in your old NeoQN app, restore it in this new version, and then you can safely delete the old app!
