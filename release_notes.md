# NeoQN (QuickNovel-Enhanced) 🚀

This is the initial release of the completely revamped, redesigned, and upgraded QuickNovel app!

## 🌟 Key Features
- **Modern UI Overhaul**: Implemented glassmorphism, beautiful gradient themes, and dynamic layout components.
- **Improved Reader Experience**: Better rendering and spacing for comfortable novel reading.
- **Modern Android Support**: App compiled natively against Android 15 (SDK 35), fully resolving legacy Android API errors, resource linking failures, and `try-with-resources` compile issues.
- **App Rebranded** from QuickNovel to **NeoQN**.

**Note**: This is an unsigned debug build, but can be fully installed on your device.
